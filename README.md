# Intro

## Idea

Minimalistic fantasy console API.

- super-simple API
- adjustable resolution
- adjustable color palette
- 1 bit sprite-sheets
- mrmotext friendly
- wav/mp3 sounds and music

## Inspiration

- Pyxel
- PICO-8
- TIC-80
- 1Bit-Wonder
- WASM-4
- Prism-384
- PQ93
- LIKO-12

https://itch.io/tools/tag-fantasy-console
